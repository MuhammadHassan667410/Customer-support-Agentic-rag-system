NOVATECH SOLUTIONS — PRODUCT FEATURES
======================================

OVERVIEW
--------
NovaTech is an all-in-one project management and team collaboration platform. It is designed to replace the fragmented combination of tools that most teams use — replacing standalone task managers, chat apps, file storage services, and reporting dashboards with one unified workspace.

This document describes every major feature available in the NovaTech platform as of the current version (v4.2).

---

TASK MANAGEMENT
---------------
The task management system is the core of NovaTech. It is built around a flexible hierarchy: Workspaces → Projects → Sections → Tasks → Subtasks.

Tasks can be created in seconds and assigned to any team member. Every task supports:
- Title, description (rich text with formatting), and due date
- Priority levels: Critical, High, Medium, Low, and No Priority
- Custom labels and tags (team-defined)
- Multiple assignees (useful for collaborative tasks)
- Time estimates and actual time logged
- File attachments (drag and drop, up to 250 MB per file)
- Comments with @mentions and emoji reactions
- Activity log showing every change made to the task with timestamps and author

Tasks can be viewed in five different layouts depending on what works best for your team:
- List view: A traditional to-do list with sorting and filtering
- Board view: A Kanban-style drag-and-drop board with customizable columns
- Calendar view: Tasks plotted on a monthly or weekly calendar by due date
- Timeline view (Gantt): Tasks displayed as horizontal bars on a timeline showing dependencies and overlaps
- Table view: A spreadsheet-style view for bulk editing and data-dense workflows

You can apply filters to any view: filter by assignee, due date range, priority, label, completion status, or any custom field.

Custom fields allow teams to track data unique to their workflow — dropdown menus, checkboxes, numeric fields, date fields, or text fields — and these appear as columns in Table view and can be used as filter criteria.

TEAM CHAT AND COMMUNICATION
----------------------------
NovaTech includes a built-in team messaging system so conversations stay connected to work.

Channels can be created for any topic, team, project, or purpose. Channels can be public (visible to all workspace members) or private (invite-only).

Direct messages support one-on-one or small group conversations. All messages support:
- Rich text formatting (bold, italic, code blocks, bullet lists)
- File and image attachments
- @mentions to notify specific people
- Emoji reactions
- Thread replies (reply to a specific message without cluttering the main channel)
- Message pinning (pin important messages in a channel for easy reference)
- Link previews for URLs shared in chat

The most powerful communication feature in NovaTech is task linking: any message can be linked to a task or a task can be created directly from a message. This means a conversation about a problem can instantly become actionable work, with context attached.

All messages are fully searchable. Search the entire workspace history by keyword, sender, date range, or channel.

---

FILE MANAGEMENT AND DOCUMENTS
-------------------------------
NovaTech provides centralized file storage and document creation so teams stop hunting through email attachments.

Files can be uploaded at three levels: the workspace level (for company-wide resources), the project level (for project-specific files), or directly attached to a task. Supported file types include all document formats, images, videos, spreadsheets, PDFs, design files (Figma, Sketch), and code files.

Storage limits:
- Starter plan: 5 GB total per workspace
- Professional plan: 100 GB total per workspace
- Enterprise plan: Unlimited storage

NovaTech Docs is the built-in document editor. Teams can create rich documents — onboarding wikis, meeting notes, project briefs, SOPs — that live inside the workspace and link to tasks and projects. Documents support real-time co-editing (multiple team members editing simultaneously), comment threads, version history (restore any previous version), and publishing as a read-only page for external sharing.

---

TIME TRACKING
--------------
NovaTech has native time tracking built into every task. Team members can:
- Start and stop a live timer on any task with one click
- Manually log time after the fact (specify date, hours, and a note)
- View their own time log across all tasks and projects in a personal dashboard

Managers and project leads can:
- View time logs for every team member across any project
- Compare estimated time vs. actual time at the task and project level
- Export time logs as a CSV for invoicing clients or running payroll

Time tracking data feeds directly into the reporting system for billable hours analysis and capacity planning.

---

REPORTING AND DASHBOARDS
--------------------------
NovaTech includes a reporting system that gives teams and managers real-time visibility into progress without needing a separate BI tool.

Built-in reports include:
- Project progress report: percentage complete, tasks overdue, tasks completed this week, upcoming deadlines
- Workload report: shows how many tasks and estimated hours each team member has assigned, helping managers spot overloaded or underutilized people
- Time tracking report: billable hours by project, by person, and by date range
- Velocity report (for teams using sprints): story points or task count completed per sprint over time
- Custom reports: build any report using any combination of data fields with a drag-and-drop report builder

Dashboards are customizable boards where teams can arrange report widgets, task lists, charts, and metrics. Each project can have its own dashboard, and there is a company-wide dashboard for leadership visibility.

Reports can be scheduled — set any report to be automatically generated and emailed to specified team members on a daily, weekly, or monthly cadence.

INTEGRATIONS
------------
NovaTech integrates with 40+ tools. Native integrations (no third-party service required) include:

Productivity and Communication:
- Slack: Receive task notifications in Slack channels, create NovaTech tasks from Slack messages, update task status from Slack
- Microsoft Teams: Same capabilities as Slack integration
- Google Workspace (Gmail, Calendar, Drive): Attach Google Drive files to tasks, sync task due dates to Google Calendar, create tasks from Gmail

Development Tools:
- GitHub: Link pull requests and commits to NovaTech tasks. When a PR is merged, the linked task is automatically moved to the "Done" column
- GitLab: Same as GitHub integration
- Jira: Two-way sync for teams that use Jira for engineering and NovaTech for the rest of the business
- Bitbucket: Link repositories and commits to tasks

Business Tools:
- Salesforce: Link CRM deals to project tasks for customer-facing teams
- HubSpot: Connect marketing campaigns to project tasks
- Zapier: Connect NovaTech to 5,000+ apps via Zapier's automation platform
- Make (formerly Integromat): Advanced automation workflows

Webhooks are available on Professional and Enterprise plans, allowing real-time event notifications to be sent to any URL when tasks are created, updated, or completed.

---

API ACCESS
----------
NovaTech provides a fully documented REST API available on Professional and Enterprise plans.

The API allows developers to:
- Create, read, update, and delete tasks, projects, and users programmatically
- Retrieve time tracking data and reports
- Manage team members and permissions
- Build custom integrations with internal tools

Authentication uses API keys (for server-to-server integrations) and OAuth 2.0 (for user-facing integrations). Rate limits: 1,000 requests per minute on Professional plan, 10,000 requests per minute on Enterprise plan.

Full API documentation is available at docs.novatechsolutions.com/api.

---

MOBILE APPLICATION
-------------------
NovaTech is available as a native mobile app for iOS (iPhone and iPad, iOS 15+) and Android (Android 10+).

The mobile app supports the full task management workflow: creating tasks, updating statuses, adding comments, uploading files from your phone's camera or photo library, and logging time. Push notifications are fully configurable — choose which events trigger notifications and at what times.

The mobile app works offline. Changes made without an internet connection are synced automatically when connectivity is restored.

The mobile app is available for free to all plan subscribers.

PERMISSIONS AND TEAM MANAGEMENT
---------------------------------
NovaTech supports granular permission management with four roles:

- Owner: Full access to everything, including billing and deleting the workspace
- Admin: Full access to all projects and settings, cannot manage billing
- Member: Access to projects they are invited to, can create tasks and comment
- Guest: Read-only or limited access to specific projects (useful for external clients or contractors)

Project-level permissions can override workspace-level roles. For example, a workspace Member can be given Admin-level access on a specific project without being promoted globally.

Enterprise customers can configure Single Sign-On (SSO) via SAML 2.0 with providers including Okta, Azure Active Directory, Google Workspace, and OneLogin.

SECURITY FEATURES
-----------------
- Two-factor authentication (2FA): Available on all plans via authenticator app or SMS
- SSO: Available on Enterprise plan
- IP allowlisting: Restrict access to specific IP addresses or ranges (Enterprise only)
- Audit logs: Full record of every action taken in the workspace — who did what and when (Professional and Enterprise)
- Data export: Export all workspace data as JSON or CSV at any time
- Data deletion: Request full account and data deletion within 30 days
- Encryption: AES-256 at rest, TLS 1.3 in transit
- EU data residency: Data hosted in AWS eu-west-1 (Ireland) region, available on Professional and Enterprise
